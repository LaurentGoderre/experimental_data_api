# Statistics Canada API Documentation #

An API for accessing Statistics Canada data. The API is built using the REST architecture and implemented using Node.JS and Postgres.

## Getting Started ##

### Prerequisites ###

* Docker (with Docker Compose)

### Installing ###

```
docker-compose up
```
